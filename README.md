# uni.tuebingen.poseidon.intro.2h.2024

This repo contains the code for a [website](https://nevrome.github.io/uni.tuebingen.poseidon.intro.2h.2024/) supporting a ~2h demonstration of Poseidon and some of its tools, features and components for students and other practitioners of archaeogenetics.

Read more about Poseidon at https://www.poseidon-adna.org.
